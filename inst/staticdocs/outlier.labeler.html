Locate Outliers in Numeric String

Usage

outlier.labeler(x, standardize = TRUE, ...)

Arguments

x
A numeric vector.
standardize
logical. If TRUE scales the vector first.
...
Other arguments passed to scale.

Value

Returns a matrix (one column) of possible outliers coded as "3sd", "2sd" and "1.5sd", corrsponding to >= to 3, 2, or 1.5 standard deviations.

Description

Locate and label possible outliers in a string.

Examples

## Not run: # outlier.labeler(mtcars$hp)# by(mtcars$mpg, mtcars$cyl, outlier.labeler)# tapply(mtcars$mpg, mtcars$cyl, outlier.labeler)# ## End(Not run)
